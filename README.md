# 서경민 개인 포트폴리오 웹사이트

QA/SQA, AI Evaluation, 자동화 테스트 학습 기록과 프로젝트를 정리하기 위한 개인 포트폴리오 정적 웹사이트입니다.

GitHub Pages를 통해 배포했으며, HTML, CSS, JavaScript만 사용해 구성했습니다.

## 배포 주소

https://dlate-1201.github.io

## 주요 구성

* **About**: QA/SQA, AI Evaluation, 자동화 테스트에 대한 관심과 학습 방향 소개
* **Projects**: 테스트 케이스, 자동화 테스트, AI 응답 평가와 관련된 프로젝트 정리
* **Notes / Blog**: QA 학습 기록, AI Evaluation 기준 정리, 투자 리서치 학습 기록
* **Contact**: 이메일과 GitHub 프로필 링크

## 사용 기술

* HTML
* CSS
* JavaScript
* GitHub Pages

## 파일 구성

```text
.
├── index.html
├── styles.css
├── script.js
└── README.md
```

* `index.html`: 웹사이트의 전체 구조와 콘텐츠
* `styles.css`: 반응형 레이아웃, 라이트/다크 테마, 접근성 스타일
* `script.js`: 모바일 메뉴, 테마 전환, 연도 표시 기능
* `README.md`: 프로젝트 설명 문서

## 주요 특징

* 별도 빌드 과정 없이 실행 가능한 정적 웹사이트
* 모바일과 데스크톱 환경을 고려한 반응형 레이아웃
* 라이트/다크 테마 전환 기능
* 시맨틱 HTML 구조 사용
* 키보드 포커스와 접근성을 고려한 스타일 적용
* GitHub Pages를 통한 간단한 배포

## 로컬에서 확인하기

별도의 설치 과정은 필요하지 않습니다.

저장소를 내려받은 뒤 `index.html` 파일을 브라우저에서 열면 바로 확인할 수 있습니다.

## 업데이트 방식

사이트 내용을 수정할 때는 아래 파일을 직접 수정합니다.

* 문구와 섹션 내용 수정: `index.html`
* 디자인과 레이아웃 수정: `styles.css`
* 메뉴, 테마 전환 등 동작 수정: `script.js`

수정 후 GitHub에 커밋하고 푸시하면 GitHub Pages에 자동으로 반영됩니다.
