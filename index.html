About

품질을 숫자와 맥락으로 설명하는 사람

저는 AI를 활용하여 테스트 자동화 설계와 AI 품질 검증을 하고자 합니다. 제품의 기능이 의도대로 동작하는지 확인하는 것뿐 아니라, 사용자가 혼란 없이 목적을 달성할 수 있는지 살피는 QA 관점을 중요하게 생각합니다.

테스트 케이스 작성, 결함 재현 절차 정리, 리스크 기반 우선순위 판단, AI 모델 응답 평가 기준 설계, 반복 검증 업무의 자동화를 중심으로 포트폴리오를 쌓고 있습니다.

  • 테스트 계획 및 체크리스트 작성
  • 버그 리포트와 재현 시나리오 문서화
  • 웹 UI 회귀 테스트 자동화 기초
  • AI Evaluation 루브릭 설계와 결과 리뷰
  • GitHub 기반 프로젝트 기록 관리
